## Bug 回報 - Reflection 提交後分數與 XP 未更新

**目前行為**
- 完成專注並提交 Reflection。
- Reward / Analysis 顯示的分數沒有明顯變化。
- 寵物 XP 進度條沒有增加或動畫不明顯。
- 點擊提交後會有約 0.x 秒停頓。

**預期行為**
- Reflection 的 `completion_percent` 應該影響本次 `quality_score`。
- 成功呼叫 `session-complete` 後，選中的真實寵物 XP 應該增加。
- Reward 應該從 `old_xp` 動畫到 `new_xp`。
- 提交後應立即顯示 saving / disabled 狀態，避免像是卡住。

- 遠端 `session-complete` Edge Function 是否已部署最新版。
- submit 時送出的 `pet_id` 是否是真實 Supabase pet UUID。
- 回傳 payload 裡的 `quality_score`、`xp_gained`、`new_xp`、`pet_id` 是否正確。
- Reward 收到的 `old_xp` 是否對應同一隻寵物。
